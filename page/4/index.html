instanceof 和 typeof 的介绍

typeof

  • 返回对应数据的类型
  • 语法：typeof xxx
  • javascript常用类型及其typeof返回值：
    • Undefined : “undefined”
    • Null : “object”
    • Boolean : “boolean”
    • Number : “number”
    • String : “string”
    • Object
      • Function object : “function”
      • Any other object : “object”
  • MDN官方说明
  • instanceof
  • 判断一个对象的prototype属性对象是否在某个构造函数的prototype链中
  • 语法：object instanceof constructor
  • MDN官方说明
